# Japanese Hiragana practice (main)
This is a small text based project that runs on if elif else print and random functions that aims to test your hiragana.

I made it because one day I got a very inspiring thought: *why spend 10 seconds finding something premade like greenowl when you can instead spend 12 painful hours making something that sucks more but still works?*

This is what set me off on this inspiring journey on pycharm.

My main struggle was that for a period of time my ctrl key broke down and was not responcive so then I had to manually edit everything. Another struggle was when I opened too many chrome tabs, the lenovo couldn't run pycharm properly so I had to restart and I almost lost all my progress. The main struggle however was that I spent so much time trying to figure out what was wrong with the code to realise that I forgot the () in .lower() . 

Run this in any platform that supports python and enjoy the code :D
